# The Mycologist – Mycelium Ops Simulation –

小さな「しいたけ職員」が森の運用を守る。
これは、業務オペレーションをモデル化した
シミュレーション＋ログ監査＋KPI分析エンジンです。

ゲーム形式ですが、中身は実務の「運用監視・可観測性・改善設計」の演習として構築しています。


---

## 🌿 Story

菌糸ネットワークに支えられた森の王国。

主人公は勇者ではなく、現場担当のしいたけ職員。
日々の業務を記録し、リスクを観測し、データで森を守る。

武器は剣ではなく「ログ」と「数値」。

集めた証拠（データ）をもとに、
腐敗や汚染の原因を可視化し、最終会議で改善を提案する。

これは「戦闘」ではなく「運用」の物語。


---

## 🎯 Purpose

現実の職場オペレーションを抽象化し、以下を定量化することを目的に設計。

- 業務負荷（HP/MP）
- 収支管理（money）
- リスク指標（汚染度・圧力・治安・士気）
- イベントログ（監査証跡）
- 日次KPI集計
- 安定性（分散）分析

感覚ではなく、
**ログとデータで状況を説明できる運用基盤の構築** を目標としています。


---

## 🏗 Architecture

状態遷移型シミュレーション + ログパイプライン構成
