# The Mycologist – Mycelium Ops Simulation 🍄

きのこ農場を舞台にした  
**業務オペレーション × シミュレーション × ログ分析のPythonプロジェクト** です。

日々の業務アクションを選択し、
状態（HP / MP / 収支）を管理しながら、
ログをCSVとして自動保存します。

ゲーム形式ですが、  
実際の業務運用・監視・改善活動をモデル化しています。

---

## 🎮 What this project does

- ターン制シミュレーション
- コマンド入力型ゲーム
- 状態管理（HP / MP / money）
- ログ自動保存（run_log.csv）
- レポート生成
- クラス設計（Player / Facility / Logger）

---

## 🧠 Skills Demonstrated

このプロジェクトで使用している技術：

- Python基礎構文
- クラス設計（OOP）
- 状態管理ロジック
- ログ設計（監査証跡）

BizOps / Operations / 自動化業務を想定した設計練習として作成。
