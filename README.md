# Operational Simulation Engine (Python)

業務オペレーションをモデル化したシミュレーション＋ログ監査＋KPIレポート生成ツールです。

日々の業務行動をイベントとして記録し、
CSVログ → 日次サマリ → KPIレポート まで自動生成することで、
「業務の可視化・安定性分析・リスク監視」を行う仕組みを実装しています。

※ ゲーム形式ですが、中身は実務オペレーション分析の設計演習です。


---

## 🎯 Purpose

現実の職場で発生する以下の要素を抽象化・定量化することを目的に設計しました。

- 業務負荷（HP/MP）
- 収支管理（money）
- リスク指標（汚染度・政治圧力・治安・士気）
- イベントログ（監査証跡）
- 日次KPI集計
- 安定性（分散）分析

感覚的な「大変さ」ではなく、
**数値とログで状況を説明できる運用基盤** を作ることが狙いです。


---

## 🏗 Architecture

Player / Facility / Event をクラスとして定義し、
状態遷移ベースでシミュレーションを実行。
